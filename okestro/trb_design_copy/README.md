## 🛠️ 개발 환경
* Node : `20.11.1`
* Vite : `5.1.4`
* Typescript : `5.2.2`
* React : `18.2.0`
* yarn berry : `4.1.0 (end date : 24.02.29)`

## 📦 Installation

```bash
# 초기 설치
yarn install
```
```bash
# 실행 명령어
# 로컬
yarn local 또는 yarn
# 개발
yarn dev
```
